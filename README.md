# Sistema de Turnos SI-3 - Documentación

## 📋 Descripción

Sistema de gestión de turnos profesional para proveedores con sincronización en tiempo real con Supabase. Permite a los proveedores solicitar turnos y a los administradores gestionar la cola de atención de manera eficiente.

## ✨ Características

### Funcionalidades Principales
- ✅ **Sincronización en tiempo real** con Supabase
- ✅ **Modo offline** - Funciona sin conexión a internet
- ✅ **Interfaz moderna y responsive** - Se adapta a cualquier dispositivo
- ✅ **Notificaciones instantáneas** - Alertas visuales para el usuario
- ✅ **Acceso seguro al panel admin** - Contraseña protegida
- ✅ **Estadísticas en tiempo real** - Turnos atendidos, en espera, proveedores
- ✅ **Historial completo** - Registro de todos los turnos atendidos
- ✅ **Gestión de proveedores** - Registro automático de proveedores

### Mejoras Implementadas

#### 1. Configuración de Supabase Mejorada
- Sistema de reconexión automática (hasta 5 intentos)

## 🚀 Instrucciones de Uso

### Para Proveedores (user.html)

1. **Acceder al sistema**
   - Abrir `index.html` en el navegador
   - Hacer clic en "Solicitar Turno"

2. **Solicitar un turno**
   - Completar todos los campos del formulario:
     - Nombre de la empresa
     - Placa del vehículo (6 caracteres)
     - Persona de contacto
     - Teléfono
     - Tipo de servicio
   - Si selecciona "Otro", describir el motivo
   - Hacer clic en "Solicitar Turno"

3. **Esperar atención**
   - Verificar su número de turno
   - Observar su posición en la cola
   - Esperar a que sea llamado

### Para Administradores (admin.html)

1. **Acceder al panel**
   - Desde `index.html`, hacer clic 5 veces en el logo
   - Ingresar la contraseña: `12345`

2. **Gestionar turnos**
   - **Llamar siguiente turno**: Atiende al próximo proveedor
   - **Completar turno**: Finaliza el turno actual
   - **Reiniciar cola**: Elimina todos los turnos en espera
   - **Sincronizar con nube**: Fuerza sincronización con Supabase

3. **Ver información**
   - Turnos en espera con opción de cancelar
   - Proveedores registrados
   - Historial de turnos atendidos
   - Estadísticas del día

## 🔧 Configuración de Supabase

### Requisitos previos
1. Crear un proyecto en [Supabase](https://supabase.com)
2. Obtener la URL y ANON KEY del proyecto

### Tablas necesarias en Supabase

```sql
-- Tabla de configuración
CREATE TABLE configuracion (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    clave TEXT UNIQUE NOT NULL,
    valor TEXT NOT NULL,
    descripcion TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT TIMEZONE('utc'::text, NOW())
);

-- Tabla de proveedores
CREATE TABLE proveedores (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    nombre_empresa TEXT NOT NULL,
    nit TEXT UNIQUE NOT NULL,
    contacto TEXT,
    telefono TEXT,
    servicio TEXT,
    activo BOOLEAN DEFAULT true,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT TIMEZONE('utc'::text, NOW())
);

-- Tabla de turnos
CREATE TABLE turnos (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    numero TEXT UNIQUE NOT NULL,
    nombre_empresa TEXT NOT NULL,
    nit TEXT NOT NULL,
    motivo TEXT,
    hora_solicitud TEXT,
    fecha_solicitud TIMESTAMP WITH TIME ZONE,
    estado TEXT DEFAULT 'espera',
    hora_llamada TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT TIMEZONE('utc'::text, NOW())
);

-- Tabla de historial
CREATE TABLE historial_turnos (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    numero TEXT NOT NULL,
    nombre_empresa TEXT NOT NULL,
    nit TEXT NOT NULL,
    motivo TEXT,
    hora_solicitud TEXT,
    hora_llamada TEXT,
    hora_finalizacion TEXT,
    estado TEXT DEFAULT 'completado',
    fecha TIMESTAMP WITH TIME ZONE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT TIMEZONE('utc'::text, NOW())
);

-- Insertar configuración inicial
INSERT INTO configuracion (clave, valor, descripcion) 
VALUES ('contador_turnos', '0', 'Contador global de turnos');
```

### Configurar credenciales

Editar `supabase-config.js`:

```javascript
const SUPABASE_URL = 'https://tu-proyecto.supabase.co';
const SUPABASE_KEY = 'tu-anon-key-aqui';
```

## 🐛 Solución de Problemas

### No se conecta a Supabase
1. Verificar que las credenciales sean correctas
2. Comprobar que el proyecto de Supabase esté activo
3. Revisar la consola del navegador para errores
4. Verificar que las tablas existan en Supabase

### Los turnos no se sincronizan
1. Hacer clic en "Sincronizar con Nube"
2. Verificar la conexión a internet
3. Revisar el estado de conexión en el panel admin

### El modal de admin no aparece
1. Asegurarse de hacer clic 5 veces en el logo
2. Esperar 2 segundos entre clics
3. Verificar que no haya errores en la consola
