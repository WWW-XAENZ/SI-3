📋 Instrucciones para Solucionar Problemas

  2. Si ves "❌ Tabla no existe", ejecuta supabase-schema.sql en Supabase SQL Editor
  3. Si ves "❌ Realtime no habilitado", ve a Database → Replication y actívalo para la tabla turnos
  4. Si ves "❌ Error de conexión", verifica que tu proyecto esté activo en Supabase
